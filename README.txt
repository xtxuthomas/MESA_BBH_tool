

Quick start:

python XuXT_MESA_model.py 1.7 0.8 1.5 2

#---------------------------------------------------------

"1.7" : log initial primary mass [Msun]
"0.8" : initial mass ratio (secondary/primary)
"1.5" : log initial orbital period [days]
"2"   : model grid number

"1" for '/vol/hal/halraid/pablo/mesa-runs/LMC_models/lowsc/models/' (LMC by Pablo)
"2" for '/vol/hal/halraid/cwang/MESA_binary_models/sc1/' (SMC by Chen)
"3" for  '/vol/hal/halraid/dpauli/GRID/' (LMC by Daniel)

#---------------------------------------------------------

This script creates pdf files. The pdf report and tex files are saved in Results/. 
See tex_template/ for my tex templates.
